# Archive Directory

This directory contains old, outdated, and deprecated files that have been archived to keep the main repository clean and organized.

## Directory Structure

### `/old-assets/`
Contains old asset files that are no longer actively used:
- `generated-icon.png` - 1.4MB generated icon file (1024x1024, archived 2025-12-12)
- `IMG_3531.png` - Old screenshot/image file (262KB)
- Various pasted text content files from development sessions

### `/deprecated-code/`
Contains deprecated code files:
- `test-deaf-first-modules.py` - Old Python test suite (superseded by Vitest test infrastructure)

### `/docs/`
Reserved for old or deprecated documentation files.

## Archive Policy

Files are archived when they are:
1. No longer referenced in active code
2. Superseded by newer implementations
3. Large files that bloat the repository
4. Historical development artifacts that may be useful for reference but not needed for active development

## Restoration

If you need to restore any archived file, you can:
1. Copy it from the archive back to its original location
2. Use git history to find the original location if unclear
3. Contact the repository maintainers for guidance
